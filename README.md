# Fredrik Thernelius's portfolio

This is my professional portfolio. It uses AWS and ReactJS.

## Build and Run Information

To run locally first install a local webserver with:
 - npm install local-web-server

After that run the following:   
   - npm install
   - npm test    (also npm run test-watch)
   - npm run webpack  <- bundles all java script files to dist/bundle.js (also npm run webpack-watch)
